# Práctica 5 - Parte 1: Simulación Física sin Interfaz Gráfica

El programa crea un entorno cerrado (la “caja”) donde varias partículas se mueven libremente, rebotan contra las paredes y pueden interactuar entre sí o con obstáculos.  
Toda la simulación se ejecuta en **tiempo discreto (Δt)** y los resultados se guardan en un archivo de texto (`resultados.txt`) para su posterior graficación.

# Práctica 5 - Parte 2: Juego de Cañones por Turnos

Este proyecto es un juego de estrategia por turnos hecho en Qt, donde dos jugadores intentan destruir la infraestructura del rival. 
Cada jugador puede ajustar el ángulo y disparar proyectiles que interactúan con el escenario y los obstáculos de forma realista. 
Las colisiones afectan la resistencia de los objetos y el juego termina cuando uno de los jugadores pierde toda su estructura.
